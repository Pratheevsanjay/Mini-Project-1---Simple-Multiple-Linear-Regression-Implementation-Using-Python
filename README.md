🧠 Algorithm Used
📈 Linear Regression

Linear Regression models the relationship between independent variable(s) and a dependent variable using a linear equation:

y=mx+b

Where:

m = slope (coefficient)
b = intercept
y = predicted value
x = input feature

In this project, the model is implemented using Scikit-Learn’s LinearRegression() class.
